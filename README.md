# Selenium E-Commerce Automation Testing

## Overview
This project automates key functionalities of an e-commerce website using Selenium WebDriver and Java.

## Tools Used
- Selenium WebDriver
- Java
- Apache NetBeans
- ChromeDriver
- GitHub

## Automated Test Cases
- Valid Login Test
- Invalid Login Test
- Add to Cart Test
- Checkout Test

## Key Features
- Browser automation
- Functional testing
- Positive and negative test scenarios
- Automated user workflow validation

## Project Structure
- LoginTest.java
- InvalidLoginTest.java
- AddToCartTest.java
- CheckoutTest.java
- Screenshots
- Test Report
